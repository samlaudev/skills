# Prompt Optimization Expert

Expert at analyzing user requirements and optimizing prompts using 57 proven AI prompt engineering frameworks. This skill helps you select the most appropriate framework based on your specific use case and generates well-structured, effective prompts.

`★ Insight ─────────────────────────────────────`
**About Prompt Frameworks**: Prompt engineering frameworks are structured templates that help organize AI requests for better results. The 57 frameworks in this collection range from simple 3-element structures (like APE) to complex multi-dimensional approaches (like RACEF), each optimized for specific scenarios such as marketing content, decision analysis, or creative writing.
`─────────────────────────────────────────────────`

## When to Use This Skill

- You want to optimize or improve an existing prompt
- You need help creating a new prompt from scratch
- You're unsure which prompt structure would work best for your task
- You want to understand different prompt engineering approaches
- You need to generate prompts for specific use cases (marketing, education, analysis, etc.)

## How This Skill Works

When you use this skill, the following process is triggered:

1. **Analyze Your Input**: You provide either a raw requirement or an existing prompt
2. **Match Best Framework**: The skill analyzes your needs against 57 frameworks to find the optimal match
3. **Clarify Ambiguities**: If your input is unclear, the skill will ask targeted questions
4. **Generate Optimized Prompt**: Based on the selected framework, a refined prompt is produced

## Quick Start

Simply describe your task or share your existing prompt:

```
"I need a prompt that helps me analyze user feedback and categorize it by sentiment"
```

The skill will:
- Identify "Sentiment Analysis" as the use case
- Match it to appropriate frameworks (e.g., TQA, GRADE, or RICE)
- Ask clarification questions if needed
- Generate a structured prompt using the best framework

## Framework Categories

### Content Creation Frameworks

- **BAB Framework** (Background, Action, Benefit): Ideal for subscription promotions and service advertising
- **BLOG Framework**: Structured blog and content marketing creation
- **RHODES Framework** (Role, Helper, Organization, Delivery, Execution, Style): For creative projects requiring high customization
- **4S Method** (Situation, Structure, Substance, Style): Content writing structure

### Decision & Analysis Frameworks

- **RICE Framework** (Reach, Impact, Confidence, Effort): Prioritization and resource allocation
- **Tree of Thoughts**: Complex multi-step problem solving
- **Six Thinking Hats**: Comprehensive decision analysis from multiple perspectives
- **Chain of Thought**: Deep logical reasoning for complex queries

### Educational Frameworks

- **Bloom's Taxonomy**: Educational content design and skill progression
- **ELI5** (Explain Like I'm 5): Simplifying complex concepts
- **Socratic Method**: Critical thinking through guided questioning
- **PEE** (Point, Evidence, Explanation): Academic writing structure

### Conversation & Assistant Frameworks

- **COAST** (Context, Objective, Actions, Style, Tone): Conversational AI design
- **ROSES** (Role, Objective, Style, Emotion, Structure): Role-play and character definition
- **TRACE** (Task, Request, Action, Context, Example): Prompt design for AI assistants

### Quick & Simple Frameworks

- **APE** (Action, Purpose, Expectation): Fast prompt construction
- **ERA** (Execute, Result, Action): Lightweight daily interactions
- **TAG** (Task, Action, Goal): Simplified task definitions
- **Zero-shot**: Quick prototyping without examples

`★ Insight ─────────────────────────────────────`
**Framework Selection Strategy**: The key to effective prompt optimization is matching complexity to your needs. Simple tasks benefit from lightweight frameworks (APE, ERA) that reduce overhead, while complex, multi-faceted tasks require structured approaches (RACEF, PROMPT) that ensure all critical dimensions are addressed. This skill automatically balances this trade-off.

## Example Usage

### Example 1: Marketing Prompt Optimization

**User Input:**
```
"I need a prompt for promoting our new eco-friendly water bottle on Instagram"
```

**Process:**
1. Identifies use case: Marketing content / Product promotion
2. Matches framework: BAB (Background, Action, Benefit) - ideal for promotional content
3. Clarifies if needed: Target audience, brand voice, key features
4. Generates optimized prompt using BAB structure

**Result:**
```
## Background
You are a social media marketing expert specializing in sustainable lifestyle products.

## Action
Create an Instagram post for our new eco-friendly water bottle. The bottle is made from 90% recycled materials, keeps drinks cold for 24 hours, and each purchase funds ocean cleanup initiatives.

## Benefit
Target audience: Environmentally conscious millennials (25-40)
Tone: Authentic, inspiring, not overly salesy
Desired outcome: Drive engagement and website visits

Include: Compelling hook, 3 key benefits, clear call-to-action, 5-10 relevant hashtags
```
